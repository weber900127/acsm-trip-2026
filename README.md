This is the itinerary planning application for the 2026 ACSM Annual Meeting trip to San Francisco, Salt Lake City, and San Diego.

## Features

- **Interactive Itinerary**: Detailed daily plans with time, location, and activity type.
- **Map Visualization**: Toggle between list and map views to see routes and locations (Leaflet + OpenStreetMap).
- **Trip Wallet**: Manage budget and expenses, syncing directly from itinerary activities.
- **Weather Integration**: Real-time weather forecasts for each city.
- **Checklist**: Track travel preparations and to-dos.
- **Admin Mode**: Edit itinerary details directly in the app (protected by authentication).

## Tech Stack

- **Framework**: React + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Map**: Leaflet / React-Leaflet
- **Animations**: Framer Motion
- **Backend**: Firebase (Firestore, Auth)
- **Deployment**: Vercel

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run development server: `npm run dev`

## Deployment

This project is configured for deployment on Vercel.
Pushing to the `main` branch should trigger an automatic deployment.
